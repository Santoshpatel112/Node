# ABES Engineering College Images

Please add the following images to this directory:

1. **abes-logo.png** - ABES Engineering College logo (recommended size: 200x50px or similar aspect ratio)
2. **campus.jpg** - Campus image for the home page
3. **student-portal.jpg** - Student portal/login image
4. **building.jpg** - College building image for the about page

## Image Guidelines:
- Logo should be in PNG format with transparent background
- Other images can be JPG or PNG format
- Recommended max width: 1000px for optimal web performance
- Ensure images are properly compressed for web use
